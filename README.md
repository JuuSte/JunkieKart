# 🏎 JunkieKart

JunkieKart is a chaotic 2D arcade racing game built with Java and FXGL.  
Designed as a local multiplayer experience for 2–4 players, the game focuses on fast-paced races, unpredictable item effects, and competitive couch gameplay.

---

## 🎮 Features

- Local Multiplayer (2–4 Players)
- Fast arcade-style racing mechanics
- Mystery item boxes with unpredictable effects
- Drug-inspired status effects with risk/reward mechanics
- Custom Main Menu UI
- Entity-based game architecture (FXGL)
- Built with Maven

---

## 💊 Gameplay

During races, players collect mystery boxes that trigger extreme temporary effects:

- **Cocaine Rush** – Massive speed boost followed by a heavy slowdown crash  
- **LSD Trip** – Visual distortion and inverted controls  
- **Alcohol Blur** – Reduced handling and blurred vision  
- **Adrenaline Shot** – Sudden acceleration spike  

Each effect has both advantages and disadvantages, adding strategy and unpredictability to every race.
